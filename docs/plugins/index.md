## 介绍

<b>自动配置插件</b>，在配置企业成员列表时需要一个一个的手动复制成员米家 ID 来配置，随着需要配置的成员越来越多，开发人员需要进行繁琐且重复的动作来完成。为了解决这个问题提高发布版本时配置测试成员的效率，开发此浏览器插件。
该插件主要包含的功能如下：<b>自动填充配置成员、对测试成员进行新增/删除/修改/查询的能力。</b>
